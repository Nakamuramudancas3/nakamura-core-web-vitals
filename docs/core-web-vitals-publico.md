# Performance e Core Web Vitals
## Nakamura Mudanças

Este documento apresenta os resultados de performance
e experiência do usuário do site Nakamura Mudanças.

## O que é analisado?
- Velocidade do site
- Estabilidade visual
- Tempo de resposta às interações

## Como é feito?
Utilizamos ferramentas oficiais do Google (Lighthouse)
para medir a experiência real do usuário.

## Resultados
Os relatórios completos estão disponíveis publicamente:

https://nakamuramudancas3.github.io/nakamura-core-web-vitals/

## Benefícios
- Melhor experiência para clientes
- Melhor SEO
- Site mais rápido e confiável
